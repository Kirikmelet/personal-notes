## 公式と公式の検証
```txt
a = 初項
r = 公比
n = 項の数
S_n = 初n項の和
```
$$S_n=a+ar+ar^2+\dots+a^{n-1}$$
$$-rS_n=-ar-ar^2-\dots-a^n$$
$$S_n-rS_n=a-ar^n$$
$$Sn(1-r)=a(1-r^n)$$
$$Sn=\frac{a(1-r^n)}{1-r}$$
