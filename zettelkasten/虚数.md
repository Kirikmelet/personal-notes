# 虚数 「キョスウ」

虚数とは数の平方根はゼロにより等しいか小さいです。

虛數是小於或等於零的數的平方根。

公式：$i=\sqrt{-1}$

## 虚数表

| 虚数  | 定数                                   |
| ----- | -------------------------------------- |
| $i$   | $\sqrt{-1}$                            |
| $i^2$ | $(\sqrt{-1})^2$, $-1$                  |
| $i^3$ | $(\sqrt{-1})^3$, $i^2 \times i$,$-i$    |
| $i^4$ | $(\sqrt{-1})^4$, $i^2 \times i^2$, $1$ |
