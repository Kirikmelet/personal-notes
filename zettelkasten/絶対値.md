# 絶対値

絶対値、それとも母数、とは実数の「$0$」からの距離です。例えば、この変数、「$x$」、は「$-1$」になりますの場合、「$x$」の絶対値は「$1$」になります。「$1$」になりますの場合も、「$x$」の絶対値は「$1$」になります。

![](2081a5ca887ae441236a175ae4a7f451e4632920.svg)

> [!info] 英語の言葉
> 
> - Absolute Value「絶対値」
> - Modulus「母数」

> $\lvert x \rvert = -x$
> $\lvert x \rvert = x$
