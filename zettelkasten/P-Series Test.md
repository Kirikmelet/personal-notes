# P-Series Test

> [!info] Theorem
> The p-series $\sum^\infty_{n=1} \frac{1}{n_p}$ is convergent if $p \gt 1$ and divergent if $p \leq 1$

> [!note] 
> Don't forget that to find the convergence use the following formula: $\frac{a}{r-1}$
