# 関数の基本的移動事

| $f(c>0)$の移動 | ｆグラフの効果      |
| -------------- | ------------------- |
| $f(x)+c$       | $c$も上垂直方向シフト |
| $f(x)-c$       | $c$も下垂直方向シフト |
| $f(x+c)$ | $c$も右方向シフト |
| $f(x-c)$ | $c$も左方向シフト |
| $cf(x)$ | $c>1$の場合、$c$単位の垂直方向の新宿。$0<c<1$の場合、$c$単位の垂直方向の圧縮|
| $f(cx)$ | $c>1$の場合、$c$単位の水平方向の新宿。$0<c<1$の場合、$c$単位の水平方向の圧縮|
