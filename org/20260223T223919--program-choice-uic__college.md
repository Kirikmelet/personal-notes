# Program Choice

Computer science was my first and only path I wanted to do. My father had a
collection of computers, ranging from Apple Mac towers running MacOS system 9;
SGI Irix computers running true commercial UNIX; custom built computers running
GNU/Linux, Windows XP/7/8/10; personal devices running archaic versions of iOS,
Android 4.1, ChromeOS when Material 1 first came out. I fell in love with computers.

In middle school in Indonesia, I looked forwards for IT classes, and I joined the robotics club
(the club did nothing), and I was one of the few who participated in an IGSCE ICT exam.
Then when I was 13, I was bored one day and I wanted to learn about "Artificial Intelligence".
So I installed VSCode, and Python 3. I self-taught myself python, and tried to follow a tutorial
to implement "Artificial Intelligence", I didn't understand what was to point and gave up.
T'was typical 13 behavior. I then decided to learn to make a "minimalist, suckless" music player in C.
So I taught myself C syntax, and had to struggle to learn about C memory mangement, differences between
pointers and references, Heap vs Stack, compilers, libraries, string management. The project went
nowhere but I had fun learning about it. I then decided to take the plunge and run GNU/Linux,
the reason was that I wanted to learn how to "hack", but installing Linux let me learn about OS internals,
how *NIX systems work and its philosophies. Later I learnt some web development, I helped my father write some
JavaScript to run in a browser, it was for an interactive website. I had to learn how version control systems worked,
how to professionally manage and test my software, and respond to my father's (client) feedback.

In high school, I made a website with 3d models embed using THREE.JS, and I made a tiny game with the Godot game engine
detailing the life of my father. My teacher applauded my work and after that I knew that this was the career I wanted.
I continued to learn C++, Rust, and other programming languages. I made countless abandoned or tiny projects. But most
importantly I learnt.

Now in college, I made an effort to learn about asynchronous programming, multi-threading,
the basic theories of machine learning and deep learning. And I am using my skills to further learn
and progress in the realm of computer science.
